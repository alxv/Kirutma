# Kirutma — Product Overview

**Version:** 1.0.0  
**Category:** Design & illustration · Desktop application  
**One-line summary:** A fast, local-first design tool for frames, shapes, and text on an infinite canvas.

---

## The problem

Designers need a focused tool that opens instantly, keeps files on their own machine, and stays out of the way while they explore layout, typography, and visual ideas. Browser tabs and heavyweight suites are not always the right fit for deep, uninterrupted work.

## The solution

**Kirutma** is a standalone desktop design application built around an infinite canvas, a familiar editor layout, and keyboard-driven workflows. Create frames, draw shapes, set type, organize layers, and export production-ready assets — without leaving the app.

---

## Core capabilities (v1.0)

### Canvas & tools

- Infinite canvas with smooth pan and zoom
- **Move**, **Frame**, **Rectangle**, **Ellipse**, **Triangle**, **Polygon**, **Star**, **Pen**, **Line**, **Text**, **Hand**, and **Scale** tools
- Multi-select, grouping, and transform handles
- Image import onto the canvas

### Panels & structure

- Layers panel with visibility and hierarchy
- Properties panel for fills, strokes, typography, opacity, and effects
- Multiple pages per document
- Color styles and text styles
- Reusable components (create, instance, detach)

### Files & export

- Native **Save** / **Open** with `.kirutma` documents
- Recent files on the home screen
- **Export PNG** at 1×, 2×, or 3× resolution
- **Export SVG** for vector handoff

### Workflow

- Full undo/redo
- Standard edit shortcuts (duplicate, copy, paste, group, ungroup)
- Keyboard shortcut reference built into the app (Help menu)
- Dark, distraction-free interface tuned for long design sessions

## Technical highlights (optional — for technical audiences)

- Native desktop shell (Tauri) for a small install footprint and OS integration
- GPU-accelerated 2D rendering pipeline (Rust + wgpu + Vello)
- Local-first documents — your work stays on your device by default

## Roadmap themes (high level)

Future direction includes richer layout systems, prototyping, and real-time collaboration. v1.0 focuses on a solid solo design experience: canvas, structure, styles, components, and export.
