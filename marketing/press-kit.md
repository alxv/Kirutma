## Boilerplate (short)

**Kirutma** is a desktop design application for frames, shapes, and text on an infinite canvas. v1.0 offers layers, styles, components, local `.kirutma` files, and PNG/SVG export.

## Boilerplate (long)

**Kirutma** is a local-first desktop design tool for UI mockups, layouts, and vector graphics. Built with a native shell and GPU-accelerated rendering, it gives designers an infinite canvas, professional panels (layers, properties, pages), reusable components and styles, and export to PNG and SVG — with documents saved as portable `.kirutma` files on the user’s device. Kirutma v1.0.0 is the first public release, focused on a fast solo design workflow.

## Feature list (copy-paste for articles)

- Infinite canvas with pan and zoom  
- Tools: Move, Frame, Rectangle, Ellipse, Triangle, Polygon, Star, Pen, Line, Text, Hand, Scale  
- Layers panel, properties panel, multi-page documents  
- Color styles, text styles, reusable components  
- Image import; fills, strokes, typography, opacity, drop shadows  
- Undo/redo; group, ungroup, duplicate; keyboard shortcuts  
- Save/open `.kirutma`; recent files on home screen  
- Export selection to PNG (1×, 2×, 3×) or SVG

## Technical notes (optional sidebar)

- **Shell:** Tauri 2 (native desktop integration, compact binary)  
- **UI:** React, TypeScript, Tailwind  
- **Rendering:** Rust, wgpu, Vello (GPU 2D)  
- **Storage:** Local `.kirutma` JSON documents (v2 bundle format planned)
