Hi everyone — we just released **Kirutma v1.0**, a desktop design tool we’ve been building for focused UI/layout work.

**What it does today:**
- Infinite canvas + standard shape/text/pen tools  
- Layers, pages, color & text styles, components  
- Save/open `.kirutma` files (local JSON documents)
