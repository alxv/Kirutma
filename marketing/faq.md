## General

### What is Kirutma?

Kirutma is a desktop design application for creating layouts and vector graphics on an infinite canvas. It includes layers, pages, styles, components, and export to PNG and SVG.

### Who is Kirutma for?

Designers, indie makers, students, and anyone who wants a local, focused tool for UI mockups and visual exploration — without mandatory cloud accounts.

## Platform & install

### Where can I download Kirutma?

Download v1.0.0 from `[DOWNLOAD_URL]`. Install by opening the provided installer or package for your system.

### Do I need an account?

No. Kirutma v1.0 works locally. Create a new file or open an existing `.kirutma` document and start designing.

### Does it work offline?

Yes. Core design, save, and export work offline once the app is installed.

## Files & format

### What is a `.kirutma` file?

A `.kirutma` file is Kirutma’s document format (JSON). It stores your pages, layers, styles, components, and embedded assets so you can reopen and share your work.

### Can I share `.kirutma` files with others?

Yes. Send the file to anyone who has Kirutma installed. They can open it from **Open file** or double-click if file association is set up.

### Will my files work in future versions?

We version the document format and aim to maintain backward compatibility. Major format changes will be documented in release notes.

## Features

### What tools are included in v1.0?

Move, Frame, Rectangle, Ellipse, Triangle, Polygon, Star, Pen, Line, Text, Hand, and Scale — plus multi-select, grouping, and transform handles.

### Can I export my work?

Yes. Select content on the canvas, then use **File → Export PNG** (1×, 2×, or 3×) or **Export SVG**.

### Does Kirutma support components and styles?

Yes. Create color styles and text styles in the Assets panel, and turn selections into reusable components with instances.

### Is there real-time collaboration?

Not in v1.0. Collaboration is on the long-term roadmap. Today, share `.kirutma` files directly.

### Are there plugins?

Not in v1.0.
